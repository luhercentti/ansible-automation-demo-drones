# Requirements para desarrollo y testing

## Sistema
- Ubuntu 22.04 LTS o superior
- Debian 11+ (compatible)
- 4 CPU cores (mínimo)
- 8 GB RAM (mínimo, 16 GB recomendado)
- 50 GB espacio en disco

## Software
ansible>=2.15.0
python>=3.8
docker>=20.10
docker-compose>=2.0

## Python Packages (instalados automáticamente)
paho-mqtt==1.6.1
requests==2.31.0
docker==6.1.3
docker-compose==1.29.2

## Puertos Requeridos
Los siguientes puertos deben estar disponibles:
- 1883  (ThingsBoard MQTT)
- 2181  (Zookeeper)
- 3000  (Grafana)
- 5683  (ThingsBoard CoAP)
- 8080  (ThingsBoard HTTP)
- 8083  (Kafka Connect)
- 8123  (ClickHouse HTTP)
- 9000  (ClickHouse Native)
- 9090  (Prometheus)
- 9092  (Kafka)
- 9100  (Node Exporter)
- 9999  (Kafka JMX)

## Versiones de Componentes
- ThingsBoard: 3.6.2
- Apache Kafka: 3.6.1 (Confluent Platform 7.5.3)
- ClickHouse: 23.12
- Grafana: 10.2.3
- Prometheus: 2.48.1
- PostgreSQL: 15

## Testing
Para verificar requisitos antes de desplegar:

```bash
# Verificar Ansible
ansible --version

# Verificar Python
python3 --version

# Verificar Docker
docker --version
docker-compose --version

# Verificar puertos disponibles
for port in 1883 8080 9092 8123 3000 9090; do
  if nc -z localhost $port 2>/dev/null; then
    echo "Port $port is IN USE"
  else
    echo "Port $port is available"
  fi
done
```
